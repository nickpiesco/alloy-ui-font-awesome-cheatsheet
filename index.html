Alloy UI Font Awesome Cheatsheet

Alloy UI 2.0 (new in Liferay Portal 6.2) ships with its own version of Font Awesome, which, though appropriately awesome, as far as I can tell doesn't quite match up with any existing Font Awesome documentation out there.

The HTML was generated by looping over the icon class names in Velocity. Check out the source if you'd like to see how it works under the hood!
